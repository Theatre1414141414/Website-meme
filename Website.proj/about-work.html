About the Project

The BATTERY-DYNAMO Mix Project is a project simply for school work in a science subject. and for showing the other students that we have potentials that they "simply" do not have.

Our work focuses on using 3D printing as "the frame" for the dynamo and the battery holder and the dynamo for squeezing and charging the elctricity into the battery.

This project serves as a work for proving that we can do things that others think is impossible.
